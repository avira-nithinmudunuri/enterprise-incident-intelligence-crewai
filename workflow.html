Agent Details

Agent 1 · Parallel

Senior IT Incident Triage Specialist

  • Classifies P1–P4 severity
  • Identifies infra / app / security category
  • Maps blast radius and business impact
  • Tools: FileRead, JSONSearch
Agent 2 · Parallel

Log Analysis & Anomaly Detection Engineer

  • Reconstructs event timeline from logs
  • Detects error spikes and cascading failures
  • Extracts stack traces and evidence
  • Tools: TXTSearch, CSVSearch, JSONSearch
Agent 3 · Sequential

Knowledge Base & Runbook Retrieval Specialist

  • Searches historical incidents via RAG
  • Finds matching runbooks and workarounds
  • Checks vendor advisories and CVEs
  • Tools: PDFSearch, WebsiteSearch
Agent 4 · Sequential

Principal Root Cause Analysis Engineer

  • Correlates triage, logs, and knowledge
  • Builds causal chain with probability scores
  • Assesses revenue, SLA, regulatory impact
  • Tools: JSONSearch
Agent 5 · Sequential

Remediation & Resilience Architect

  • Immediate mitigation (< 1 hour)
  • Short-term fixes (< 24 hours)
  • Long-term preventive investments
  • Tools: WebsiteSearch
Agent 6 · Sequential

Communications & Reporting Director

  • Technical incident report
  • Executive summary for leadership
  • Lessons learned and process improvements
  • Outputs: report.md / .html / .pdf
